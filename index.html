05 宽禁带半导体与核心技术Wide Bandgap & Core Technologies

⚡ SiC 碳化硅

1200V SiC MOSFET 用于 Totem-Pole PFC，导通电阻低至33mΩ，开关频率>100kHz。比传统Si方案效率提升2-3%，计划2027导入1700V SiC。

1200V SiC MOSFET for Totem-Pole PFC, RDS(on) 33mΩ, switching >100kHz. 2-3% efficiency gain vs traditional Si. 1700V SiC planned for 2027.

🔮 GaN 氮化镓

650V GaN HEMT 用于辅助开关和同步整流，零反向恢复损耗。TO-LL封装支持高密度集成。2028目标：双向GaN单级变换。

650V GaN HEMT for auxiliary switches and sync rectification, zero reverse recovery. TO-LL package for high-density. 2028 goal: Bi-directional GaN single-stage.

🧲 磁集成技术

PCB平板变压器替代传统绕线，磁集成LLC变压器+谐振电感一体化。矩阵变压器方案支持4000W+，大幅缩减体积和损耗。

PCB planar transformers replace traditional wire-wound. Integrated LLC XFMR + resonant inductor. Matrix XFMR for 4000W+, reducing volume and losses.

🧩 模块化设计

借鉴AcBel和Great Wall的模块化策略：PFC/LLC/Aux独立功能块，跨CRPS/CR80/Slim共用核心模块。减少50%开发周期，提升SMT自动化率。

Inspired by AcBel & Great Wall: Independent PFC/LLC/Aux blocks, shared across CRPS/CR80/Slim form factors. 50% faster dev cycle, higher SMT automation.

🖥️ 全数字控制

DSP (TI C2000 / STM32G4) + CPLD协控平台。PMBus 1.2+全数字监控，支持FW OTA在线升级。Blackbox故障记录≥15条，动态响应<100μs。

DSP (TI C2000 / STM32G4) + CPLD co-processor. PMBus 1.2+ full digital monitoring, FW OTA upgrade. Blackbox ≥15 records, dynamic response <100μs.

🔄 供电架构演进

12V → 48V → 54V → 800V。54V已成AI GPU主流，800V HVDC为下一代方向。支持AC+DC UPS、48V BBU、SST HVDC多种架构。

12V → 48V → 54V → 800V. 54V is now AI GPU mainstream, 800V HVDC is next-gen. Supports AC+DC UPS, 48V BBU, SST HVDC architectures.
